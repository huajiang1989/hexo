我会一步一步带领大家进入 Vue2 的世界，拥抱变化，熟悉 Vue 1.x 的根据文档可以迅速掌握 2.0，因为其中大约 90% 的语法是重复的。2.0 更多是基于框架本身的优化，整体设计思想是不变的。
